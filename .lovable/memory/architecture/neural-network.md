The network functions as a learning organism governed by a 9-layer Instinct Hierarchy (localSecurity → networkSecurity → connectionIntegrity → consensus → torrentTransfers → decentralization → exploration → creativity → coherence). Layer 3 (connectionIntegrity) gates upper layers via its connectionHealth signal.

**Layer suppression is continuous attenuation, never a hard cut.** When a lower layer's health dips below `STABILITY_THRESHOLD` (0.5), upper layers are *quieted* by a multiplicative factor `min(1, h_low / 0.5 + 0.25)` — never silenced. Floor: `ATTENUATION_FLOOR = 0.15`. A layer is considered `active` when `health ≥ 0.3`. This preserves smooth evolution under `𝒪_UQRC` — no discrete cliffs that drop terms out of the master equation. Lower-layer degradation is logged but no longer flips upper layers to `active = false`.

The engine uses Welford-derived bell curves to track interaction stats and Φ phase transitions to summarize coherence. See `mem://features/network-entity` for how stages are derived from this state.

**Field coupling.** The neural engine reads real `Q_Score`, basins, `curvatureMap`, and `dominantWavelength` from `getSharedFieldEngine()` (`src/lib/uqrc/fieldEngine.ts`). Replaces the synthetic `m2/100` curvature proxy at the old line ~749 of `neuralStateEngine.ts`. Every `onInteraction()` calls `field.inject(peerId, { reward, trust })`, so peers are lattice sites. `getPeerScore()` divides the trust contribution by `1 + getCurvatureForText(peerId)`. Decay heartbeat is `clamp(60 s, dominantWavelength × 1500 ms, 15 min)`. After 3 consecutive snapshots inside a basin a peer is `pin()`-ed at 1.0. Failure-retry kind suggestion goes through `selectByMinCurvature` (log-only). See `docs/BRAIN_UNIVERSE.md` › "Neural ↔ Field coupling".

**Learning manifold ↔ Field coupling.** `PatternLearner`, `LanguageLearner`, and `DualLearningFusion` read curvature/basins/wavelength from the same shared field. `PatternLearner.ingestEvent` injects every event (`event.type`, reward, trust) into the lattice. `DualLearningFusion.computeReward` multiplies engagement by `1 / (1 + getCurvatureForText(event.text))` — high-curvature text earns less. `EXPLORATION_RATE` is now derived: `clamp(0.02, 1/(1+λ), 0.25)`. `selectPattern` consults `selectByMinCurvature` over all matching patterns when ≥ 2 match. `LanguageLearner` auto-pins top tokens that stay basin-resident for ≥ 3 ingestions (FIFO cap 64). Phrase merges require `count ≥ 5` AND `isTextInBasin(bigram)` once the field is warmed up. Cold-start uses count-only rule. Coupling visible via `[Learning↔Field] Q=… pinnedTokens=… explore=…` log (≤ 1/5 s).
